Features

Built for how you work with Claude.

Claude writes and reads a lot of markdown. Canto gives you the dedicated tools to manage that context without leaving your IDE.

✏️

WYSIWYG editor

Edit markdown in the rendered view. Bold, italic, headings, tables — real typography, not raw syntax.

🧠

Memory browser

All your .claude/memory/*.md in one place, with types, stale indicators, and one-click create.

📋

Plans view

See plans from docs/superpowers/plans/ or .claude/plans/. Scaffold new ones instantly.

⚙️

Config panel

Quick read-only view of your MCP servers, permissions, and hooks from Claude Code settings.

📄

CLAUDE.md dashboard

Open CLAUDE.md from the sidebar. Sections and rules rendered as cards, not walls of text.

🎨

Theme-aware

Matches your VS Code dark/light theme automatically. Looks native everywhere.
